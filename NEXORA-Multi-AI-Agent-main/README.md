# NEXORA AI 🧠

An advanced AI workspace with intelligent agents.

## Features

- Research Agent 🔎
- Knowledge Agent 📚
- Planner Agent 📅
- Voice Interaction 🎙️
- PDF Generation
- AI Chat Interface

## Tech Stack

- Python
- Streamlit
- OpenAI API

## Run

Install dependencies:

pip install -r requirements.txt

Run:

streamlit run app.py
